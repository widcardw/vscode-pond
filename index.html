VSCode Pond

A playful fish pond built with VS Code icons

This project renders VS Code logo icons as "fish" swimming across a deep ocean scene using PixiJS v8. It includes a water ripple distortion effect and an animated wave surface overlay.

  • Inspired by the Show Aquarium feature in VS Code Agent
  • Reference implementation by youyoumu
  • Built with PixiJS, Vite, and TypeScript
